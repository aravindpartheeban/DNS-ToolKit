# DNS Toolkit

**DNS Toolkit** is a lightweight Windows utility that helps you test, compare, and switch DNS providers using a simple, developer-friendly interface. It supports a range of gaming, privacy-focused, ad-blocking, and family-safe DNS providers, and allows you to benchmark latency and apply a selected DNS profile to your network adapter with a single click.

---

## Features

- DNS latency benchmarking using parallel ping tests  
- One-click DNS switching for your selected network adapter  
- Built-in recommendation engine:
  - Gaming
  - Privacy
  - Security
  - Family-Safe
  - Balanced
- Displays your currently configured DNS servers
- Uses native Windows networking tools only
- No telemetry or data collection

---

## Recommendation Engine

If you’re unsure which DNS provider to choose, DNS Toolkit can recommend one automatically based on your goal — such as:

- gaming performance  
- privacy  
- security  
- ad-blocking  
- family-safe filtering  

The app measures latency (where applicable), filters relevant providers, selects a suitable option, and explains the reasoning.

---

## Supported DNS Providers

### Gaming / General Use
- Cloudflare  
- Google  

### Privacy-Focused
- NextDNS  
- Quad9  
- Mullvad  
- ControlD  

### Ad-Blocking
- AdGuard  
- Mullvad Adblock  
- ControlD (Hagezi & OISD profiles)

### Family-Safe DNS
- CleanBrowsing Family  
- AdGuard Family  
- Quad9 Security  
- ControlD Family  

Each provider includes a short information popup describing its purpose.

---

## System Requirements

- Windows 10 or Windows 11  
- Administrator privileges required to change DNS settings  
- Distributed as a portable executable (no installer required)

---

## Privacy Notice

DNS Toolkit does **not** proxy DNS traffic and does **not** collect or transmit analytics or diagnostic data.  
All configuration changes are applied locally using standard Windows networking commands.
